# CIS-121 Lab 2: Basic C++ Logic & Arithmetic

This repository contains three C++ programs focused on basic input/output, arithmetic operations, and data types.

## 🚀 Projects Included

### 1. Coin to Cents Calculator
**Path:** `/Coin-Calculator`
- **Purpose:** Converts a user-defined number of quarters, dimes, and nickels into a total cent value.
- **Concepts:** Integer arithmetic and constants.

### 2. Sales Receipt Generator
**Path:** `/Receipt-Generator`
- **Purpose:** Calculates the subtotal and final total (including 7% tax) for a purchase.
- **Concepts:** `double` precision, `iomanip` for currency formatting, and constant variables.

### 3. Temperature Converter (Celsius to Rankine)
**Path:** `/Temperature-Converter`
- **Purpose:** Converts Celsius temperatures to Fahrenheit, then to Degrees Rankine.
- **Concepts:** Floating-point math and multi-step conversion formulas.

## 🛠️ Requirements
- A C++ compiler (like G++, Clang, or Visual Studio)
- Standard C++ Libraries (`iostream`, `iomanip`, `math.h`)
